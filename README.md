# Whattodo

#### Todo앱

<br/>

## 개발 인원 및 기간

- 개발기간 : 4주
- 개발 인원 : 프론트엔드 1인

<br/>

## 목적

<br/>

#### 알람의 목적에 맞게 프로필을 사용하여 효과적인 관리를 위한 Todo앱

<br/>

## 적용 기술 및 구현 기능

<br/>

### 적용기술

- Html
- Css
- Vue.js
- Express.js
- Mongodb
- Aws Elastic Beanstalk

<br/>

### 구현 기능

- Express.js 와 Mongodb 연결
- Aws Elastic Beanstalk 로 서버배포
- Mongodb에 DB 셋팅 및 Aws Elastic Beanstalk 서버 연결

<br/>

#### Owner

- Today(당일 날짜 이동)
- Calendar
- Settings(프로필이미지 및 이름 변경)
- 알림
  - 제목,시간,내용 설정
  - 알림기능
